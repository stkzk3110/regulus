# README

Railsチームの開発アプリです。

開発に参加する手順

```bash
$ git clone git@github.com:yasasii-team/regulus.git
$ cd regulus
$ bundle
$ rails db:create
$ rails db:migrate
$ git checkout -b <new_branch>
```
